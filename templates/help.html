Arkad Wealth FAQ everything you need to know




This section contains most of the information needed for using Arkad Wealth If you cannot find an answer to your question make sure to contact us.

Customers

When can I invest with Arkad Wealth

Our website is currently used only for researchers and investors, but stay tuned for future updates

What does the Customer version of Arkad Wealth do ?

For now Arkad Wealth for customers provides limited functionality, we've used the indexes ( SPY, IEI, LQD , QQQ) To generate the 3 Portfolios that customers can choose ( Read disclaimer first)
Investors

What stocks can I use?

Any stock that is listed in Yahoo! Finance is available for use if you are not sure use this link

What dates can I use ?

When choosing a ticker, make sure the dates you are choosing are valid ( Meaning the stock was out between those dates)
e.g if I want to use Arkad wealth to get all the data for FB I would first look at the IPO of FB and choose dates from that date until today (May 18, 2012 simply googling FB IPO)

How many stocks can I use?

Minimum 2 stocks and for now Arkad wealth allows only 10 maximum stocks for building your portfolios

What does the graph the was generated means?

The Graph generated by Arkad Wealth shows a distribution of all the different portfolios that were simulated using different weights for each stock, the 3 most interesting stocks are highlighted in Green Yellow and red
  • Green - Is the highest Sharpe ratio Portfolio Read more about sharpe ratio
  • Yellow- Is the safest portfolio that was generated by Arkad wealth meaning it's the portfolio which had the lowest risk between the dates selected
  • Red - Is the portfolio which had the highest returns between the dates that were selected
Developers

How does Arkad Wealth work?

We use Python as the backend application to run the calculations that are used in generating the graphs and finding the optimal portfolios. We used the following libraries :
  • Yfinance for pulling the financial data for each stock
  • Matplotlib for generating the graph
  • Pandas For financial data manipulation
