Arkad Wealth - A Financial Robot Advisor

Sapir College exLab - Fintech Team

A Financial Tool for Investors and Researchers

Arkad Wealth uses Python to Create and simulate portfolios using random weights for each stock,

Then it selects the 3 most interesting Portfolios, Highest return Portfolio, Safest volatility portfolio, and highest sharpe ratio portfolio

For now the results can be used by Investors getting more information, and in the near future we plan to create a highest scale robot.
